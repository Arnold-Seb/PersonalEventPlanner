# Personal Event Planner App

## Overview
This Android application allows users to manage events by creating, viewing, updating, and deleting them. It uses Room database for local storage and Jetpack Navigation for smooth user experience.

## Features
- Add new events
- View upcoming events sorted by date
- Edit event details
- Delete events
- Local storage using Room database
- Fragment-based navigation
- Input validation and error handling

## Technologies Used
- Kotlin
- Android Studio
- Room Database
- RecyclerView
- ViewModel & LiveData
- Jetpack Navigation Component

## How to Run
1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run the app on emulator or device
